Mobile app with AI backend

Daily 60

Daily 60 turns one user goal into a seven-step plan, then breaks it into daily actions, reminders, reflections, and streak progress. The important work was keeping the mobile app simple while making the AI response predictable.

  • Moved prompt construction, provider keys, JSON extraction, Zod validation, quotas, and fallbacks into the backend.
  • Built the app with Expo Router, Zustand, MMKV persistence, local notifications, haptics, Render deployment, EAS profiles, and Vitest coverage.

Finance dashboard

FinTrack

FinTrack is a small-business finance dashboard built around the work that happens before the chart looks clean: imports, review states, categories, budgets, recurring costs, and accountant exports.

  • Built CSV import with duplicate detection, column mapping, vendor category memory, import summaries, and flagged transaction review.
  • Added Supabase Auth wiring plus user-owned tables, indexes, triggers, RLS policies, reports, budgets, and review records.

Shopify ecommerce storefront

Kakebo Shop

Kakebo Shop is a live Shopify storefront where the work centered on catalog discovery, collection pages, product presentation, and enough SEO structure for niche collectibles to be findable.

  • Worked across Liquid sections, snippets, collection templates, navigation, visible SEO content, FAQ schema, related links, and storefront QA.
  • Used Shopify Admin GraphQL scripts for product clarity, collection updates, SEO fields, and catalog maintenance.

Local service website
